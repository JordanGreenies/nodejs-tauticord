Simple, zero bloat, nodejs Discord bot for tautulli, displaying all the current streams, quality and bitrate.

## Docker instructions:
```
git clone https://github.com/JordanGreenies/nodejs-tauticord
cd nodejs-tauticord
vi config.json
docker-compose up
```
